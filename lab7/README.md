# Module 7 Lab: Simulated Serverless Python Function

This lab simulates a serverless Python function in Codio. The function runs on port 8000, accepts an optional name parameter, and returns a JSON greeting with the current time.

## Run the Function

Command: python3 function.py

## Test the Function

Command: curl "http://localhost:8000/?name=Michael"

The function was also accessed through the Codio port 8000 URL. One challenge was saving the code in the wrong folder, which I fixed by copying the completed file into the lab7 directory.
