# Lab 5 – File Storage and Retrieval

This lab simulated file storage for an ecommerce image catalog in Codio.

## Completed Tasks
- Created local storage folders for uploads, catalog images, thumbnails, and featured products.
- Created two sample product image files.
- Copied the images into upload and catalog directories.
- Installed MySQL and created a database named `mydatabase`.
- Created an `images` table to store image names and file paths.
- Inserted records for Product 1 and Product 2.
- Moved Product 1 into a featured folder and updated its database path.
- Created a backup archive of the storage directory.

## Final Image Paths
- Product 1: `catalog/featured/product1.svg`
- Product 2: `catalog/product2.svg`

## Files
- `images.sql` contains the SQL commands used to create the database table, insert sample records, and update Product 1 after it was moved.
- `assets/` contains the two sample SVG product images used in the lab.
